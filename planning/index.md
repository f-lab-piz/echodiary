# 플래닝 인덱스

## 현재 방향
- Ideation 1번 항목인 `키워드/자연어 기반 일기 자동 작성`을 1차 MVP 주제로 선택함.

## 중요성
- 일기 작성 진입 장벽을 낮춰 기록 지속성을 확보하는 핵심 기능임.

## MVP 범위
- 키워드/자연어 입력 기반 일기 생성.
- 계정 단위 페르소나 보유 + 일기장 단위 다중 연결/전환.
- 페르소나 생성(템플릿/수정/LLM 대화) 및 이미지 생성 재시도 정책.

## 활성 스펙
